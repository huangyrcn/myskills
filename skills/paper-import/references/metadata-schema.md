# metadata.yaml Schema

`metadata.yaml` is the single source of truth shared across all steps.
It lives at `res/{identifier}/metadata.yaml`.

## Identifier Format

`{author}{year}{keyword}` — generated by `scripts/query_apis.py`

```
Attention Is All You Need (Vaswani, NeurIPS 2017) → vaswani2017attention
BERT (Devlin, NAACL 2019)                         → devlin2019bert
Deep Residual Learning (He, CVPR 2016)             → he2016deep
```

## Full Schema

```yaml
# === Written by scripts/query_apis.py (Step 1) ===

title: "Attention Is All You Need"
authors:
  - "Ashish Vaswani"
  - "Noam Shazeer"
year: 2017
venue: "NeurIPS"

identifiers:
  arxiv: "1706.03762"           # present if found on arXiv
  doi: "10.5555/3295222.3295349"
  semantic_scholar: "..."
  openalex: "..."
  openreview: "..."             # present if ICLR/NeurIPS/ICML
  pmc: "..."                    # present if PubMed Central

urls:
  arxiv_abs: "https://arxiv.org/abs/1706.03762"
  doi: "https://doi.org/10.5555/3295222.3295349"
  s2: "https://semanticscholar.org/paper/..."

abstract: |
  We propose a new simple network architecture, the Transformer...

pdf_urls:
  - source: arxiv
    url: "https://arxiv.org/pdf/1706.03762"
    reliability: high           # high / medium / low
  - source: openreview
    url: "https://openreview.net/pdf?id=..."
    reliability: high

latex_source:
  available: true               # true ONLY for arXiv papers
  url: "https://arxiv.org/e-print/1706.03762"

# === Appended by get-paper after download (Steps 2-3) ===

assets:
  pdf: paper/paper.pdf          # always present after Step 2
  markdown: paper/paper.md      # always present after Step 2 (markitdown)
  latex_dir: paper/             # present only if latex_source.available
  repo: repo/                   # present only if code found in Step 3
```

## Key Field Notes

| Field | Source | Used by |
|-------|--------|---------|
| `identifiers.arxiv` | query_apis.py | PDF download, LaTeX download |
| `identifiers.doi` | query_apis.py | Papers with Code fallback |
| `pdf_urls[].url` | query_apis.py | PDF download (try in order) |
| `latex_source.available` | query_apis.py | whether to attempt LaTeX download |
| `latex_source.url` | query_apis.py | LaTeX tar download |
| `assets.*` | get-paper | downstream consumers (lit-review, etc.) |

## reliability levels

- `high` — direct open-access URL, almost always works (arXiv, OpenReview, PMC)
- `medium` — usually works but may redirect (SSRN, CORE, Unpaywall)
- `low` — may be paywalled or redirect to landing page (DOI, S2, OpenAlex)

Always try `high` sources first; only fall back if the download fails file verification.
